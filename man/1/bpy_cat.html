bpy_cat(1)

Name

bpy cat - print the contents of a list of files to stdout

Synopsis

cat print the contents of a list of files to stdout, concatinating them into a single stream. Useful for quickly inspecting the contents of a single file.

Usage

bpy cat [-when=TIMESPEC] path

Example

Read the contents of two files:

$ bpy ls
hello.txt
world.txt
$ bpy cat hello.txt world.txt
hello
world!
